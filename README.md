## Development

Uses:

- [asdf](https://asdf-vm.com)

```bash
# Install asdf dependencies
source <(curl -sL https://raw.githubusercontent.com/olmesm/odd-scripts/main/shell/asdf-install.sh)

# Install dependencies
npm install

# Start dev server
npm run dev

```
